# Memory Game JS

## Tecnologias Utilizadas

- HTML
- CSS
- Javascript

## Objetivo

O usuário deverá selecionar duas cartas, se as cartas forem iguais, as cartas ficam fora do jogo.
O jogo termina quando o usuário acertar todas as cartas.
